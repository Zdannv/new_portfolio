# **App Name**: Zaidan's Digital Showcase

## Core Features:

- Project Highlights: Interactive showcase of Muhammad Zaidan Zhafiz Satrianto's key projects, focusing on visual appeal and detailed project descriptions.
- Cleanova Circle Details: Detailed view for the Cleanova Circle project, highlighting the full-stack web development, Next.js, and Supabase implementation with dynamic video progress tracking.
- Adibasa App Showcase: Showcase the Adibasa Android app, focusing on the UI/UX design process and user-centered design implementation for learning Javanese.
- StreetWare Platform Presentation: Present the StreetWare app with emphasis on geolocation-based reporting and real-time notifications for e-government solutions using Flutter and Supabase.
- VehiCare Functionalities Overview: Provide an overview of the VehiCare app, demonstrating its GPS tracking capabilities and maintenance scheduling features.
- Technical Skills Visualization: A dynamic display of technical skills using a tag cloud to highlight proficiency in mobile and web development, cybersecurity, and UI/UX design.
- Leadership Skills Showcase: Visually represent leadership skills and experiences, highlighting roles in Open Source Community PENS, Efortech, Student Company SMA Negeri 1 Waru, and HIPMI PT PENS.
- Achievements & Competitions Display: Display achievements and competition results, including JA Titan Business Competition, Jawa Pos SMA Awards, Gemastik, and other participations.
- Contact Interaction: Show a call to action button that will send an email.

## Style Guidelines:

- Dark, professional background (#121212) for an elegant dark mode feel.
- Primary color: Deep purple (#6200EE) to evoke innovation and modernity, offering strong contrast on the dark background.
- Accent color: Electric blue (#3F51B5) to draw attention to CTAs and important information, positioned analogously to the primary for visual harmony.
- Body text: 'Inter' sans-serif for a clean and modern look. Headline font: 'Space Grotesk' for headings to establish a balance between modern and personable aesthetics.
- Minimalist icons with a line-art style to maintain a modern aesthetic and ensure clarity. Color: use a slightly lighter color than primary, for increased visibility
- A grid-based layout system to ensure responsiveness across devices. Project cards will utilize high-resolution images with subtle hover effects.
- Subtle transitions and animations to enhance user experience without being distracting. Consider loading animations and smooth scrolling effects.
